# Brightzy — Android Application

A large-scale, modular Android application built for Bave Holdings LLC (USA).
Live on Google Play Store with 400+ active paid users.

🔗 [View on Google Play Store](https://play.google.com/store/apps/details?id=com.brightzy.app)

## 📱 About the App

Brightzy is a feature-rich Android application offering interactive maps, 3D components, and real-time data experiences. The app serves 400+ active paid users and is maintained in a production environment with regular Play Store releases.

## 🏗️ Architecture Overview

- **15+ feature modules** — each module independently developed and maintained
- **100+ fragments** — complex navigation flows managed across the application
- **MVVM architecture** — clean separation of concerns across all modules
- **Unity integration** — native Android to Unity bridge for interactive maps and 3D components
- **WebSocket layer** — real-time bidirectional communication for live data updates
- **REST API integration** — backend data synchronization and content management
